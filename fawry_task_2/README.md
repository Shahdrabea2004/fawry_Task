Quantum Bookstore – Fawry N² Dev Slope #10 Challenge
A Java system for managing and selling books by type:
Paper books, eBooks, and Showcase books.

![image](https://github.com/user-attachments/assets/51f3e866-92fe-4883-a101-7e8183160054)


All console outputs start with:
Quantum book store:

Project Description
Add books to inventory.

Remove books older than a specified number of years.

Purchase paper and electronic books with stock checks and delivery methods.

Showcase books are not for sale.

Designed with an extensible OOP approach to easily add new book types without modifying existing code.

How to Run
Download or copy the file QuantumBookstoreFullTest.java.

Open it in any Java IDE (e.g., IntelliJ IDEA or VS Code).

Run the program and observe the console output.

Supported Features
Add books with details: ISBN, title, publication year, price, author.

Remove books older than a certain number of years.

Purchase books by ISBN and quantity, send eBooks via email, ship paper books.

Showcase books cannot be purchased.

Project Structure
Book (abstract base class)

PaperBook

EBook

ShowcaseBook

QuantumBookstore (manages inventory and operations)

QuantumBookstoreFullTest (main class for running tests)


All operations print with prefix Quantum book store:.
